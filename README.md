# 中国語学習管理アプリ

## 概要
中国語講読の授業で扱った作品を、本文・ピンイン・日本語訳・メモと一緒に保存し、復習時に見返せるWebアプリです。

## 作成背景
授業ごとのプリントや翻訳メモが散らばり、試験前に復習しづらい課題があったため、自分の学習記録を一元管理できるツールとして作成しました。

## 使用技術
HTML / CSS / JavaScript / localStorage

## 機能
- 作品登録
- 中国語本文登録
- ピンイン登録
- 日本語訳登録
- メモ登録
- 作品一覧表示
- 作品削除
- localStorageによるデータ保存
